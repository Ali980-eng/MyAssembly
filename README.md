# MyAssembly

This repository contains the assembly programs I develop using **EMU8086** as part of my current college coursework.  
It includes various examples and practice files related to x86 assembly programming.

## Contents
- **Learning**: Basic instructions, simple programs, and code snippets used to understand assembly concepts.
- **Sandbox**: Experimental code, testing ideas, and trying out new instructions or techniques.
- **Assignment**: Course-related tasks, homework, and required programming exercises.

## Requirements
- **EMU8086** or any compatible x86 assembler/emulator

## Notes
These files are created for academic and personal learning purposes.  
Feel free to explore and review the examples included.
